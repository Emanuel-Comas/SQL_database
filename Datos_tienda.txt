Ejercicio: Crear una base de datos y realizar varias operaciones en ella

Parte 1: Crear una base de datos y una tabla:

Crear una base de datos llamada tienda.
Crear una tabla llamada productos con las siguientes columnas:

id: un número entero, clave primaria, autoincremental.
nombre: un texto con el nombre del producto.
precio: un número decimal con el precio del producto.
categoria: un texto que describe la categoría del producto.
stock: un número entero que indica la cantidad de productos disponibles.


Parte 2: Insertar datos en la tabla:

Insertar al menos 5 productos en la tabla productos. Utiliza diferentes valores para 
los campos nombre, precio, categoria y stock.



Parte 3: Consultar datos:

Mostrar todos los productos de la tabla productos.
Mostrar solo los productos que pertenecen a la categoría 'Electrónica'.
Mostrar los productos cuyo precio sea mayor a 50.
Mostrar solo los nombres y precios de los productos.



Parte 4: Actualizar y eliminar datos:

Actualizar el stock de un producto. Por ejemplo, 
aumenta el stock del producto con id 1 en 5 unidades.
Eliminar un producto de la tabla, por ejemplo, elimina el producto con id 3.



Parte 5: Consultas adicionales:

Mostrar los productos ordenados por precio, de mayor a menor.
Contar cuántos productos hay en total.
Mostrar el precio promedio de los productos.



Objetivo del ejercicio:

Este ejercicio permitirá practicar las siguientes operaciones SQL:

Crear bases de datos y tablas.
Insertar datos en una tabla.
Realizar consultas simples usando SELECT.
Filtrar resultados con WHERE.
Ordenar resultados con ORDER BY.
Actualizar y eliminar datos con UPDATE y DELETE.
Usar funciones agregadas como COUNT y AVG.
